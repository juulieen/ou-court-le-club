Mentions legales

Editeur
Julien OLLIVIER, membre du club Run Event 86. Ce site est un projet personnel et n'est pas une initiative officielle du club.

Donnees personnelles
Ce site affiche les prenoms des membres du club inscrits a des courses, avec leur consentement. Aucun nom de famille n'est publie. Les prenoms ne sont pas stockes dans le code source du site ; ils sont generes a la volee lors du deploiement.

Les donnees sont collectees a partir de listes d'inscription publiques de plateformes de courses a pied, dans le but d'animer la vie du club. Base legale : consentement (art. 6.1.a RGPD).

Droit d'opposition
Tout membre peut demander le retrait de son prenom a tout moment. Conformement au RGPD, vous pouvez exercer vos droits (acces, rectification, suppression, opposition) en contactant l'editeur a runevent86@juulieen.fr.
