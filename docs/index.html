Mentions legales

Editeur
Julien OLLIVIER, membre du club Run Event 86. Ce site est un projet personnel et n'est pas une initiative officielle du club.

Donnees personnelles
Ce site affiche uniquement le nombre de membres du club inscrits a des courses. Aucun nom ni donnee personnelle n'est publie sur ce site.

Les donnees sont collectees a partir de listes d'inscription publiques de plateformes de courses a pied, dans le but d'animer la vie du club.

Droit d'opposition
Conformement au RGPD, vous pouvez exercer vos droits (acces, rectification, suppression, opposition) en contactant l'editeur a runevent86@juulieen.fr.
